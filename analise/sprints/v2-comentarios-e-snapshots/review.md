# Review - V2 Comentarios e Snapshots

## Perguntas Obrigatorias

1. Comentários ancorados funcionam de forma confiável no fluxo principal?
2. Snapshots realmente preservam momentos relevantes do board?
3. Restaurar uma versão anterior é previsível e seguro?
4. A solução continua compatível com a simplicidade do MVP?
5. Os objetivos desta versão continuam bem definidos ou precisam de ajuste?

## Se o Objetivo Nao Foi Atingido

Registrar:

1. se a falha foi de modelagem, UX, persistência ou integração com canvas;
2. se o vínculo por `shapeId` precisa de nova abordagem;
3. se a versão deve ser dividida ou ter objetivos revisados.

## Decisao de Saida

Escolher uma conclusão explícita:

1. `aprovada para seguir`
2. `aprovada com ressalvas`
3. `nao aprovada para seguir`
